Looking at the code, the issue is in `_DescriptorBase.__get__`. When `obj` is `None` (which happens when accessing the descriptor from the class level, e.g., `MyClass.get`), the code tries to call `self.Wrapper(obj)` with `None`, which then fails.

The fix is to return the descriptor itself (or some appropriate object) when `obj is None`, so class-level inspection works correctly.
